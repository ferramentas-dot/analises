# Análises — Swipeoffers

Repositório de análises publicadas via GitHub Pages.

## Análises disponíveis

- [Razzetti](razzetti/) — Reels + Feed (Instagram), YouTube e tráfego via SimilarWeb.

## Estrutura

```
.
├── index.html          # Landing com links pras análises
├── razzetti/
│   ├── index.html      # Análise renderizada
│   ├── frames/         # Thumbnails de reels (IG)
│   ├── feed/           # Imagens de posts de feed (IG)
│   ├── youtube/thumbs/ # Thumbnails YouTube
│   └── similarweb_prints/  # Prints SimilarWeb
```

## Publicação

Site é servido via GitHub Pages a partir da branch `main`, root `/`.
